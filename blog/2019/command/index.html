CommandBuffer

Aug 16, 2019 • Huailiang

Command Buffer是Unity5新增的一个非常强大的功能。在渲染的时候，给OpenGL或者DX的就是一系列的指令，比如glDrawElement，glClear等等，这些东西目前是引擎去调用的，而Unity为我们封装了更高一级的API，即CommandBuffer，可以让我们更加方便灵活地实现一些效果。CommandBuffer最主要的功能是可以预定义一些列的渲染指令，然后将这些指令在我们想要的时机进行执行。

提取Mesh

如图所示，在$VTX之后的字符*.npki 20个字节之后表示的是子网格顶点个数6D-00-00-00（绿色部分), 反向字节排序0x006D = 109即十进制是109。

在$RSI之后的28个字节，我们有顶点vertex起始地址60-D7-01，并且在它下面，在13个字节之后，表示indices索引起始地址E0-FC-01, 十六进制表示即为0x1fce0。
